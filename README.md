# Practice Coding

Coding practice in **JavaScript**, **Python**, and **C++**.

- One folder per problem  
- Shared test cases  
- Scripts to install dependencies and run tests

## Setup
```bash
./scripts/install-deps.sh
```
Python script to take the problem name and create .py and .js files. 
## Test
```bash
./scripts/run.sh <problem-name>
```
